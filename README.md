# RISCV-Pipelined-SoC-FPGA
This Project is a complete documentation of step by step incremental implementation of a RISCV Pipelined SoC on Terasic De10-Nano FPGA Board. The system includes a RISCV pipelined Core, HPS based compilation of C program to FPGA's BRAM as executable for soft core. Button Synchronizer System, SPI Controller and Hardware based LCD Driver.
